# 🌍 GeoAI & Spatial Machine Learning Portfolio
**Philipps-Universität Marburg | MO:VE Exchange Program (Summer 2026)**

## Overview
This repository documents my applied research and engineering implementations in bridging **Remote Sensing** and **Machine Learning** to drive data-driven spatial decisions. It contains spatial data pipelines, algorithmic implementations, and models developed during my coursework at the Department of Geography (Lab of Environmental Informatics), Marburg University.

- **Focus:** Automating spatial workflows, integrating LiDAR with Sentinel data, and deploying scalable Spatial ML models.

## 🗂️ Repository Structure
The repository is structured around three core architectural pillars:

- `📁 01_Base_R`: Core R programming, spatial data wrangling, and foundational data structures.
- `📁 02_Remote_Sensing_LiDAR`: Machine Learning applications (e.g., Random Forest), spatial predictions, and handling multi-source remote sensing data.
- `📁 03_Advanced_SDM`: Advanced Species Distribution Modeling, generating virtual species, and critically evaluating environmental predictors.

##  Research Motivation & Extensions
Beyond the standard curriculum deliverables, my objective is to explore the spatial transferability of these models. I aim to test how algorithms trained on European datasets can be adapted to arid environments and complex agricultural systems, such as the Nile Delta in Egypt.
